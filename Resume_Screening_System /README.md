# 📄 Resume Screening System (Python Project)

## 📌 Project Overview
The Resume Screening System is a Python-based mini project designed to simulate how real-world Applicant Tracking Systems (ATS) filter and rank candidates based on job requirements.

It compares a candidate's resume with a job description and calculates a **match percentage score**, highlighting matched and missing skills.

---

## 🚀 Features
- Extracts skills from resume text
- Compares resume with job description
- Calculates match percentage score
- Identifies matched and missing skills
- Generates a CSV report for results
- Simple and beginner-friendly implementation

---

## 🧠 How It Works
1. Load job description from text file  
2. Load candidate resume from text file  
3. Extract predefined skills using Python logic  
4. Compare skills between resume and job description  
5. Generate:
   - Match Score
   - Matched Skills
   - Missing Skills  
6. Save results into a CSV file
